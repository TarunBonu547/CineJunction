13

Design Tokens & Developer Notes

These tokens and conventions should become the single source of truth for future UI work.

Core tokens

  • Colors: primary, background, surface, text, secondary, accent states
  • Typography: logo, display, hero, title, UI, body, caption
  • Spacing: 4px–96px scale
  • Radius: 4px, 8px, 12px, 16px, 18px, 24px, 999px
  • Shadows: small, medium, large, hover, hero

Implementation notes

  • Use semantic HTML and ARIA labels in every component.
  • Keep keyboard navigation and visible focus states intact.
  • Prefer variables over hard-coded values for scalable theming.
  • Test across 480px, 768px, 1280px, and 1440px breakpoints.
  • Maintain cinematic contrast while preserving readability.
